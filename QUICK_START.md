# K-JULIA Quick Start Guide

## 3 Simple Steps to Run

### Step 1: Test the System
```bash
python3 test_system.py
```
Expected output: "ALL TESTS COMPLETED SUCCESSFULLY!"

### Step 2: Run Interactive Demo
```bash
python3 main.py
```
This starts the dashboard and interactive mode.

### Step 3: Open Dashboard
Open your browser to: http://localhost:5000

## What to Try

In the interactive mode, type:
- "Power cut in our area" - Files a power complaint
- "Water supply issue" - Files a water complaint  
- "sample" - Adds sample data to dashboard
- "exit" - Quits the program

## What You'll See

1. **AI Responses**: K-Julia will respond naturally to your complaints
2. **Reference Numbers**: Each complaint gets a unique ID (e.g., KJ12345)
3. **Database Logging**: All complaints are stored in Supabase
4. **Dashboard**: Real-time analytics and visualizations

## System Architecture

```
User Input → AI Engine → Response + Database Log → Dashboard Display
```
